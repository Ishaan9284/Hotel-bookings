# Hotel-bookings
📊 Project Overview

This project performs Exploratory Data Analysis (EDA) on hotel booking data to identify patterns related to booking cancellations, pricing trends, seasonal demand, and customer behavior.

The analysis helps understand factors affecting hotel reservations and provides insights that can support data-driven decision making in the hospitality industry.

⸻

🎯 Project Objectives

The main goals of this project are:
	•	Analyze hotel booking patterns
	•	Identify factors influencing booking cancellations
	•	Study Average Daily Rate (ADR) trends over time
	•	Compare booking behavior between City Hotels and Resort Hotels
	•	Understand seasonal demand across months
	•	Generate insights through data visualization
🛠 Tools & Technologies Used
	•	Python
	•	Pandas – Data manipulation
	•	NumPy – Numerical analysis
	•	Matplotlib – Data visualization
	•	Seaborn – Statistical visualization
	•	Jupyter Notebook – Development environment

⸻

🔎 Project Workflow

1️⃣ Data Cleaning
	•	Removed unnecessary columns
	•	Handled missing values
	•	Converted date columns into datetime format

2️⃣ Exploratory Data Analysis (EDA)

Key analyses performed:
	•	Hotel booking trends
	•	Cancellation rate analysis
	•	Monthly booking demand
	•	ADR (Average Daily Rate) trends
	•	Customer demographics
	•	Comparison of City Hotel vs Resort Hotel bookings

3️⃣ Data Visualization

Visualizations used to highlight patterns:
	•	Cancellation trends
	•	ADR trends over time
	•	Monthly demand patterns
	•	Hotel type comparisons
	•	Booking distribution

⸻

📈 Key Insights

Some insights obtained from the analysis:
	•	City hotels experience higher cancellation rates than resort hotels
	•	Hotel pricing (ADR) varies significantly across different seasons
	•	Certain months show higher booking demand
	•	Lead time strongly influences cancellation probability
